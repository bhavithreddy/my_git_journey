**Example Use Case**

Here is an example of how to use MCP in a Python project:
```python
import docker

# Initialize the MCP context
mcp_context = docker.MCPContext()

# Establish a connection with the model
model = mcp_context.connect_model("my_model")

# Use the MCP protocol to send data to the model
mcp_context.send_data(model, {"input": "example data"})
```
